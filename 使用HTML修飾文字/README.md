橄欖醬（橄欖醬）

這道菜做起來非常簡單，在聚會上總是很受歡迎。 我父親建議：

“在前一天晚上做這個，這樣味道就有時間混合。在你上菜之前把它調到室溫。在冬天，試著把它加熱。”

原料

1 8 盎司。 罐裝曬乾的西紅柿
2個大蒜瓣
2/3 攝氏度 卡拉馬塔橄欖
1噸。 刺山柑

指示

在食品加工機中混合西紅柿和大蒜。 攪拌至盡可能光滑。

加入酸豆和橄欖。 脈衝電機幾次，直到它們合併，但仍保留一些
質地。

搭配山羊奶酪和新鮮羅勒裝飾（可選）的薄烤麵包片。
