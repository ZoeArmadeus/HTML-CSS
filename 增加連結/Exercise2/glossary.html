詞彙表

A

別名
在位圖圖形的顏色之間可能出現的鋸齒狀階梯狀邊緣。
錨點
負責創建超鏈接的 HTML 元素。
小程序
一個自包含的小型可執行程序，例如用 Java 編程語言編寫的程序。
屬性
擴展或修改 HTML 元素的參數

B

後端
與服務器上發生的功能有關的網站方面
位深度
基於文件或系統分配的位數的顏色測量
瀏覽器
請求和顯示網頁的客戶端軟件。

C

緩存
瀏覽器用於下載文檔的臨時存儲區域
計算機生成
Common Gateway Interface，Web服務器與服務器上運行的其他程序之間進行通信的一種機制
編解碼器
媒體文件用於壓縮和解壓縮數據的算法
壓縮
一種減小文檔文件大小的方法。 不同的文件格式使用不同的壓縮技術

D

抖動
通過混合圖像或系統調色板中可用的相似顏色的像素來近似顏色。
文檔類型定義 (DTD)
為標記語言定義每個元素和屬性的 XML 文檔。
域名
與特定IP地址對應的名稱。

E

外聯網
在封閉系統內可用的網站或系統，通常通過密碼選擇系統外的用戶。

F

閃光燈
一種用於在 Web 上傳送動畫、交互和音頻的多媒體格式
幀率
在視頻和動畫中，每秒的幀數
FTP
文件傳輸協議； 一種通過 Internet 將文件從一台計算機移動到另一台計算機的系統

G

伽瑪
指計算機顯示器顯示的整體亮度。
動圖
圖形交換格式； Web 圖形圖像的通用文件格式。 GIF 是一種基於調色板的 8 位格式。

H

十六進制
使用字符 0-9 和 A-F 的 base-16 編號系統。 它在 CSS 和 HTML 中用於指定顏色值
主機
服務器的另一種說法。 託管服務是為網站提供服務器空間的公司。
HTML
超文本標記語言； 用於識別網絡文檔結構的標記語言。
HTTP
超文本傳輸協議； 定義如何在服務器和瀏覽器之間請求和傳輸網頁和媒體的系統。

I

圖像映射
包含多個超文本鏈接的單個圖像
IP地址
網絡上計算機或設備的數字標識符。
網絡服務提供商
互聯網服務提供商； 一家銷售互聯網計算機網絡訪問權的公司

J

Java
Sun Microsystems 開發的一種跨平台、面向對象的編程語言。
JavaScript
一種腳本語言，可以為網頁和瀏覽器上的元素添加行為。

L

有損
一種壓縮類型，其中丟棄數據以減小文件大小。 JPEG 格式使用有損壓縮方案。

M

MP3
一種流行的高質量音頻文件格式，使用 MPEG 壓縮
MPEG
由電影專家組制定的一系列多媒體標準

N

嵌套
將一個 HTML 元素放置在另一個元素中。 HTML 和 XHTML 定義了哪些元素可以嵌套在另一個元素中的規則。

O

優化
減小文件大小以縮短網絡傳輸時間。

P

調色板
8 位索引顏色文件中的表格，為圖像中的像素提供顏色信息。
路徑名
使用目錄層次結構和文件名由斜杠分隔的命名法指向文件
像素
位圖圖形圖像中的單個正方形
PNG
便攜式網絡圖形； 一種圖形文件格式，支持 8 位和 24 位圖像

Q

QuickTime
Apple Computer開發的一種適用於所有平台的音頻和視頻格式

R

分辨率
位圖圖像或計算機顯示器中每英寸的像素數 (ppi)
RGB 顏色
一種基於紅、綠、藍光組合來描述顏色的顏色系統
根目錄
網站的頂級目錄。 根目錄包含站點的所有文件和子目錄。

S

無襯線字體
一種字體設計，具有直線或方形字符筆劃。 Helvetica 就是一個例子。
襯線字體
一種在字符筆劃末端有水平板的字體。 時代就是一個例子

T

補間
動畫工具中的一個函數，用於在開始幀和結束幀之間自動創建幀。

U

Unix
由貝爾實驗室開發的多用戶、多任務操作系統。
網址
通用資源定位器； 萬維網上文檔或資源的地址

V

矢量圖
使用數學方程定義形狀和填充的圖形。 矢量圖像可以在不改變質量的情況下調整大小。

W

W3C
萬維網聯盟； 為 Web 開發製定通用標準的公司和組織聯盟。
網絡調色板
一組 216 色，可在運行 8 位彩色顯示器的 Mac 和 Windows 計算機上使用。

X

XHTML
根據XML規則編寫的新HTML標準。
XML
可擴展標記語言； 一種用於創建自定義標記語言的元語言
